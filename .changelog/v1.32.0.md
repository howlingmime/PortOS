# Release v1.32.0

Released: 2026-03-12

## Added
- CoS agent task descriptions now render as formatted markdown
- LLM gate system for autonomous jobs — configurable approval gates before agent execution
- Per-task unblock buttons on blocked tasks banner for quick task management
- Task metadata for agent options (worktree, simplify) on scheduled tasks
- Mobile-friendly task add form styling

## Changed
- Markdown files optimized — fixed stale references, deduplicated content, consolidated audits

## Fixed
- Git branch delete now reactively updates the UI instead of full page reload
- POST LLM drills route correctly during loading and support text answers in element flash
- PortOS self-build no longer causes socket hangup; UI shows graceful toast on connection loss
- Unblocking tasks from the banner reactively updates both the banner and task list without page reload
- Various null guards, hooks violations, and review feedback fixes
